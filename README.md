# 部署一次，手机永久用（不用开电脑）

## 为什么之前要开电脑？

`.bat` 只是在你家 WiFi 里临时搭一个小服务器，**不是正规用法**。

真正做法是：**网页免费放到网上 → 手机装一次 → 以后直接点图标**。

数据（房源、照片、租期）都在 **手机本地**，不依赖电脑。

---

## 最简单步骤（GitHub，约 15 分钟）

### 1. 注册
打开 https://github.com → Sign up → 登录

### 2. 新建仓库
右上角 **+** → **New repository** → 名字填 `fangdong` → **Create**

### 3. 上传文件
点 **uploading an existing file** → 把 `new ider` 文件夹里这些内容拖进去：

- `index.html`
- `css/` 文件夹
- `js/` 文件夹
- `icons/` 文件夹
- `manifest.json`
- `sw.js`

（`启动租房管家.bat` 不用上传）

点 **Commit changes**

### 4. 开启网站
**Settings** → 左侧 **Pages** → Source 选 **main** → **Save**

等 1～3 分钟，得到地址：

```
https://你的GitHub用户名.github.io/fangdong/
```

### 5. 手机安装（只做一次）
- 手机浏览器打开上面 **https** 地址
- **添加到主屏幕**
- 以后只点桌面 **「租房管家」**，**不用开电脑**
